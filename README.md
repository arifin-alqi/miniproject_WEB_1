# Website Portfolio

## Deskripsi Project
Project ini merupakan website portfolio sederhana yang dibuat menggunakan HTML, CSS, Bootstrap 5, dan Vue JS.  
Website terdiri dari tiga section utama yaitu Home, About Me, dan Certificates.

## Penjelasan Code

### 1. HTML
- Menggunakan struktur dasar HTML (html, head, body)
- Navbar menggunakan Bootstrap
- Grid system Bootstrap digunakan untuk layout
- Vue JS digunakan untuk menampilkan data skills dan certificates

### 2. CSS
- Styling background setiap section berbeda
- Menggunakan min-height: 100vh untuk fullscreen section
- Hover effect pada card
- Responsive menggunakan Bootstrap breakpoints

### 3. Vue JS
- Menggunakan Vue.createApp()
- Data disimpan dalam data()
- Menggunakan v-for untuk menampilkan skills dan certificates
- Menggunakan interpolation {{ }}
